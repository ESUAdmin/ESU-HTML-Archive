恶俗维基:隐私政策

  • 其他语言
  • 监视本页
  • 编辑

为了使访客或注册用户（以下简称“用户”）能够更好的使用恶俗维基（以下简称“本站”），本站可能需要收集一些个人信息，本隐私策略用于让用户知晓本站收集哪些信息，以及本站如何处理用户的个人信息。

目录

GDPR 豁免

根据欧盟数据保护通用条例（GDPR）的规定，本站的运营主体为个人，因此本站无需遵守 GDPR，也无需发布数据处理单位的身份。

数据处理单位

名称：无

地址：无

联系方式：本站首页发布的联系方式

收集信息的方式

  1. 用户向本站提交的内容，例如，用户在编辑框中输入了内容，并点击提交按钮
  2. 网站的自动化设施，例如 Javascript 和服务器日志记录

使用信息的方式

  1. 本站使用服务器日志记录，以确保本站的信息安全
  2. 本站使用部分个人信息，以确保本站的内容免遭破坏
  3. 本站使用 Google Analytics 生成的统计信息，以分析网站访客来源等
  4. 在评论区匿名发表评论，并提及他人或使用回复功能时，被提及/回复的人将得到您的 IP 地址
  5. 在评论区违反评论规则时，您的 IP 地址将公开出现在封禁列表中
  6. 部分情况下，本站管理员可能会使用收集到的信息，但不会分享该信息

分享信息的方式

除本页面所述之外，本站不会分享收集到的相关信息。

第三方内容

  1. 本站使用了 Google Analytics 以收集统计信息，关于 Google Analytics，请参阅 Google 隐私政策
  2. 本站使用了 Google reCAPTCHA 以保护网站免受自动化程序的破坏，关于 Google reCAPTCHA，请参阅 Google 隐私政策
  3. 本站使用了 CloudFlare 作为内容分发网络，关于 CloudFlare，请参阅 CloudFlare 隐私政策
  4. 本站可能包含外部链接，本站不对外部链接的内容、安全性或信息收集等行为负责

用户权利

  1. 用户有权通过复制等操作获取用户发布的信息副本，根据版权策略，用户发布内容代表向本站永久授权，因此，本站可能不会删除用户发布的信息
  2. 用户有权删除其发布的内容，但是如果要删除内容，应当经过所有贡献过该页面的用户同意
  3. 用户有权请求删除其他用户发布的内容，但应当经过所有贡献过该页面的用户同意
  4. 用户有权删除其帐号
  5. 其他法律赋予用户的权利

数据的删除和变更

  1. 当本站上的内容被删除或修改后，本站保留还原该内容的权利，本站不保证也无法保证其他用户或第三方网站将原内容进行存档
  2. 本站无条件执行真实的法院判决结果，但本站无法做到的除外。此外，为了保护用户信息，对于获取用户个人资料的请求，本站只处理来自法院判决的该类请求
  3. 本站可能会执行来自执法机构的命令，除非明确要求保密，否则无论本站是否执行该命令，都会将该请求的内容公开
  4. 本站会按实际情况酌情处理来自用户的请求，例如删除内容，修改用户信息，纠正不实信息等。请注意，删除内容请求只有当符合下列条件之一时才会执行：
    1. 大部分用户同意删除该内容
    2. 所有编辑过该页面的人同意删除该内容
    3. 违反美国法律的内容
    4. 违反恶俗维基守则的内容
    5. 其他特殊情况

数据保护

  1. 本站采用 TLS 加密技术以确保数据传输的安全
  2. 本站采用 Web 应用防火墙（WAF）以降低网站受到黑客攻击的可能性
  3. 本站严格遵循金融行业数据安全标准（PCI DSS）规范并已进行评估：
    1. 本站使用多重安全服务和防火墙
    2. 本站使用了加密技术进行数据存储和传输
    3. 本站为所有系统执行定期检查，并确保操作系统或软件不存在已知漏洞
    4. 本站严格限制服务器的管理权限，并拥有安全审计日志
  4. 本站无法保证也不可能保证数据的绝对安全

技术细节或详细信息

  1. 本站采用 Cookie 技术以分析用户的访问情况，或保存用户的登录会话，关于 Cookie 以及 Cookie 如何工作，请参阅 Cookie
  2. 本站的服务器会自动记录日志文件，日志文件包含以下信息
    1. 用户的 IP 地址
    2. 反向代理服务器的 IP 地址
    3. URL（网址）
    4. 来源 URL
    5. 时间
    6. 用户代理字符串（User-Agent）
  3. 当用户向本站主动提交内容时，例如在输入框中输入内容并点击提交按钮，服务器可能会记录任何用户主动提交的信息
